系統設定

🔑 Claude API Key
1
申請 API Key（免費額度可用）
前往 console.anthropic.com → 登入或註冊 → 點「API Keys」→「Create Key」
2
複製 Key 貼入下方
Key 儲存在您的瀏覽器本機，不會上傳到任何地方。每份查核紀錄解析費用約 NT$1–3 元。
ℹ 費用說明
· 每份查核紀錄約 2,000–5,000 字，解析費用約 NT$1–3 元
· Anthropic 新帳號有免費額度（約 $5 USD），可解析約 50–100 份紀錄
· 之後按用量計費，一年查核紀錄若有 20 份，全年費用約 NT$40–60 元
· 委員學經歷搜尋請在 Claude.ai 對話視窗操作後複製填入，不消耗 API 費用
備份與還原
定期備份，避免瀏覽器清除 localStorage 後資料遺失
📤 匯出備份
將所有資料匯出為 JSON 檔，建議每次匯入新查核紀錄後備份一次。
📥 還原備份
選擇備份 JSON 檔還原，將覆蓋現有資料。
⚠ 清除所有資料
清除瀏覽器中的所有查核紀錄、委員與缺點資料，此操作無法復原。
